# moto-o-meterAI
O Moto-o-MeterAI é uma evolução do moto-o-meter para identificar as melhores oportunidades de compra da Kawasaki Versys-X 300 no mercado brasileiro, onde usamos I.A Local para análise da descrição para identificar se a moto possui Sinistro ou Passagem de Leilão e analisando se o valor é atrativo
